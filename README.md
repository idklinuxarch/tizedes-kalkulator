# Tizedes Tört Kalkulátor

Magyar nyelvű tizedes tört kalkulátor általános iskolásoknak (5-6. osztály).

## Funkciók
- Tizedes törtek ábrázolása számegyenesen
- Kerekítés
- Összeadás, kivonás
- Szorzás/osztás 10-zel, 100-zal, 1000-rel
- Szorzás/osztás természetes számmal
- Átlag kiszámítása
- Tört → tizedes tört átalakítás
